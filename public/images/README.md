# Image Assets for Hero Component

## Required Images

Place the following images in this directory:

### 1. zurich-poster.jpg
- **Purpose**: Video poster/placeholder image
- **Dimensions**: 1920x1080 (or similar 16:9 ratio)
- **Format**: JPEG, optimized for web
- **File size**: < 500KB recommended

### 2. zurich-fallback.jpg
- **Purpose**: Fallback if video fails to load
- **Dimensions**: 1920x1080 (or similar 16:9 ratio)
- **Format**: JPEG, optimized for web
- **File size**: < 500KB recommended

## Where to Find Free Zurich Images

### Unsplash
https://unsplash.com/s/photos/zurich
- High-quality, free to use
- Search: "zurich aerial" or "zurich cityscape"

### Pexels
https://www.pexels.com/search/zurich/
- Free stock photos
- Good selection of city views

### Pixabay
https://pixabay.com/images/search/zurich/
- Free for commercial use
- Various Zurich city shots

## Image Optimization Tips

1. Use JPEG format (better compression for photos)
2. Resize to exactly 1920x1080 before uploading
3. Compress using:
   - https://tinyjpg.com/
   - https://squoosh.app/
   - ImageOptim (Mac)
4. Target final size: 200-500KB
